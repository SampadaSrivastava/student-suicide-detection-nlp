# student-suicide-detection-nlp
# Student Suicide Detection using NLP

## Overview

This project uses Natural Language Processing (NLP) and Machine Learning to classify student-written text into:

* Suicidal
* Non-Suicidal

The goal is to assist in early mental health risk detection by analyzing textual data.

## Dataset

* Total Samples: 5000
* Classes:

  * Suicidal
  * Non-Suicidal

## Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* NLTK
* Scikit-Learn

## Exploratory Data Analysis

* Missing Value Analysis
* Label Distribution
* Text Length Analysis
* Word Count Analysis
* Word Cloud
* Bigram Analysis

## Machine Learning Pipeline

Text Cleaning → TF-IDF Vectorization → Logistic Regression → Prediction
